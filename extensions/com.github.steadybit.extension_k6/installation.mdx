# Prerequisites

<Accordion title="Helm / Kubernetes">
  To install the chart with the name <code>steadybit-extension-k6</code>.

  <Code lang="bash" code={`
helm repo add steadybit https://steadybit.github.io/extension-k6
helm repo update

helm upgrade steadybit-extension-k6 \
  --install \\
  --wait \\
  --timeout 5m0s \\
  --create-namespace \\
  --namespace steadybit-extension \\
  steadybit/steadybit-extension-k6`} />

If you want to use K6 Cloud you need to provide an k6 cloud api token. You can add it for example with <code>--set k6.cloudApiToken="111-222-333"</code>

</Accordion>
<Accordion title="Docker">
  You may alternatively start the Docker container manually.

  <Code lang="bash" code={`
docker run \\
  --env STEADYBIT_LOG_LEVEL=info \\
  --expose 8087 \\
  steadybit/extension-k6:latest
`} />
</Accordion>
