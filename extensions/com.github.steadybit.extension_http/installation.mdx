<Accordion title="Helm / Kubernetes">
  To install the chart with the name <code>steadybit-extension-http</code>.

  <Code lang="bash" code={`
helm repo add steadybit-extension-http https://steadybit.github.io/extension-http
helm repo update

helm upgrade steadybit-extension-http \\
  --install \\
  --wait \\
  --timeout 5m0s \\
  --create-namespace \\
  --namespace steadybit-extension \\
  steadybit-extension-http/steadybit-extension-http`} />

</Accordion>
<Accordion title="Docker">
  You may alternatively start the Docker container manually.

  <Code lang="bash" code={`
docker run \\
  --env STEADYBIT_LOG_LEVEL=info \\
  --expose 8085 \\
  --name steadybit-extension-http \\
  ghcr.io/steadybit/extension-http:latest
`} />
</Accordion>
